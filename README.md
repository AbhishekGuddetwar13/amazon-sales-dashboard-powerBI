# Amazon Sales Store Dashboard

## Project Overview

This project presents an interactive Power BI dashboard developed using Amazon sales data. The dashboard provides insights into sales performance, profit analysis, order trends, customer segments, product categories, shipping methods, and regional sales distribution.

The objective of this project is to transform sales data into meaningful business insights that support strategic decision-making.

## Business Insights

### Highest Sales Region
The dashboard allows comparison of Central, East, South, and West regions to identify top-performing markets.

### Best Performing Category
Office Supplies generated the highest sales contribution among all categories.

### Top Profitable Sub-Category
Copiers delivered the highest profit among all sub-categories.

### Customer Segment Analysis
Consumer customers contributed the largest share of total sales.

### Return Analysis
More than 95% of orders were successfully completed without returns.
